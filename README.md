# Trading Telegram Bot

Телеграм-бот для закрытого трейдинг-канала:

- утром присылает черновик приветственного поста;
- делает карточку "новости на сегодня";
- ищет сетапы по крипте, рисует чистый график с входом / стопом / тейками;
- отправляет все сначала владельцу на согласование;
- хранит последние посты, чтобы новые тексты были логичнее;
- принимает обучающие файлы и использует их как базу знаний;
- умеет работать с несколькими AI-провайдерами с fallback;
- готов к деплою на Railway.

## Что внутри

- `python-telegram-bot` для Telegram-бота и джобов
- `SQLite` для черновиков, истории постов и сигналов
- `Binance` public API для свечей и мониторинга цены
- `RSS` + `CoinGecko` для новостного фона
- локальная генерация картинок через `matplotlib` и `Pillow`
- LLM fallback: `OpenRouter` -> `Groq` -> `Gemini` -> локальный шаблонный режим

## Быстрый старт

1. Создай бота через `@BotFather`.
2. Добавь бота в свой канал и дай ему права на публикацию.
3. Скопируй `.env.example` в `.env` и заполни переменные.
4. Установи зависимости:

```bash
pip install -r requirements.txt
```

5. Запусти:

```bash
python main.py
```

## Команды и кнопки

В личке с ботом у владельца будут кнопки:

- `Тест: доброе утро`
- `Тест: новости`
- `Тест: сигнал`
- `Черновики`
- `Статус`
- `Помощь`

## Как работает публикация

1. Бот генерирует черновик.
2. Черновик уходит тебе в личку.
3. Ты жмешь `Одобрить`, `Редактировать` или `Отклонить`.
4. Только после `Одобрить` пост публикуется в канал.

## Обучение бота

Можно присылать боту файлы:

- `.txt`
- `.md`
- `.pdf`
- `.docx`

Бот вытаскивает текст, складывает в локальную базу знаний и потом подтягивает подходящие куски в промпт.

## Railway

Для Railway достаточно:

- `Procfile`
- `runtime.txt`
- переменных окружения из `.env.example`

Стартовая команда:

```bash
python main.py
```

## Важно

Это контентный и аналитический бот, а не автотрейдинг. Логика сигналов здесь эвристическая: бот ищет аккуратные сетапы по тренду и пишет посты, но финальная проверка остается на тебе.
